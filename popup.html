1. 행 컨테이너 셀렉터

한 항목(row)을 감싸는 요소의 CSS 셀렉터 (예: .product-list > li)

2. 추출할 필드

각 행 내부에서 추출할 필드들. 셀렉터는 행 기준 상대 경로입니다. 비워두면 행 자체.

3. 크롤링 범위

URL이 안 바뀌는 SPA(jQuery bootpag, React 등)에서 페이지 번호를 한 칸씩 클릭. 예시 사이트의 data-lp: #page-selection li[data-lp="{N}"] a

📊 구글시트 연동 (선택)

Apps Script로 배포한 웹앱 URL과 토큰을 입력하면 결과를 시트에 추가/불러오기 할 수 있습니다. 셋업 방법은 apps-script.gs 파일 또는 README 참조.
