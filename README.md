# MJ Collect — 크롬 웹 크롤링 확장프로그램

CSS 셀렉터로 페이지 안의 항목과 필드를 지정해 데이터를 수집하고, 표 / CSV / JSON / TSV(구글시트 붙여넣기용)로 내보내는 크롬 확장입니다.

## 설치 (개발자 모드 로드)

1. 크롬 주소창에 `chrome://extensions` 입력
2. 우측 상단 **개발자 모드** 켜기
3. **압축해제된 확장 프로그램 로드** 클릭
4. 이 폴더(`mjcollect`)를 선택

설치 후 툴바의 **MJ** 아이콘을 클릭하면 팝업이 열립니다.

## 사용 방법

### 1) 행 컨테이너 셀렉터
한 항목(상품, 게시글 등)을 감싸는 요소의 CSS 셀렉터를 입력합니다.
- 예: `ul.product-list > li`, `.board .post`, `article.item`

크롬 개발자도구(F12)에서 요소를 우클릭 → **Copy → Copy selector** 로 빠르게 가져올 수 있습니다(다만 자동 생성 셀렉터는 너무 길거나 깨지기 쉬워, 클래스 기반으로 정리하는 게 안정적입니다).

### 2) 추출할 필드
각 행 내부에서 뽑을 필드들을 정의합니다.

| 칸 | 의미 |
| --- | --- |
| 필드명 | 결과 표의 컬럼 헤더가 됩니다. (예: `제목`, `가격`) |
| 상대 셀렉터 | 행 기준의 CSS 셀렉터. 비워두면 행 자체. (예: `.title a`, `.price`) |
| 추출 방식 | `text` / `html` / `href` / `src` / `value` / 속성명 직접 |

`속성명 직접`을 선택하면 우측 칸에 `data-id`, `alt` 같은 임의의 속성명을 입력할 수 있습니다.

### 3) 크롤링 범위
- **현재 탭만** — 지금 보고 있는 페이지에서 한 번 추출 (가장 안전)
- **페이지 범위 (URL 패턴)** — `https://example.com/list?page={N}` 처럼 `{N}`을 포함한 URL을 시작~끝 번호로 자동 순회
- **페이지 범위 (다음 버튼)** — 페이지 안의 "다음" 버튼 셀렉터를 클릭하며 최대 N페이지까지 (SPA/무한페이지 일부에 적합)

### 4) 결과 / 내보내기
- 팝업 하단에 표 형태로 즉시 표시됩니다.
- **TSV 복사** — 구글시트 / 엑셀에 그대로 Ctrl+V 가능
- **CSV** — 엑셀 한글 호환 BOM 포함 다운로드
- **JSON** — 파일 다운로드

### 프리셋
사이트별로 자주 쓰는 설정을 이름 붙여 저장 / 불러올 수 있습니다.
설정은 모두 브라우저 로컬에 저장되며 외부로 전송되지 않습니다.

## 알려진 제약 사항

- **무한 스크롤** 사이트는 미지원 (다음 버튼이 있는 사이트는 가능).
- **자바스크립트로 늦게 렌더링되는 컨텐츠**는 지연(ms) 값을 늘려 보세요.
- `chrome://`, `chrome-extension://` 등 크롬 내부 페이지에서는 동작하지 않습니다.
- **구글시트 직접 연동(API)**은 1차 버전에서 미포함 — `TSV 복사` → 시트에 붙여넣기로 사용해 주세요.

## 파일 구조

```
mjcollect/
├── manifest.json     # Manifest V3
├── popup.html        # 팝업 UI
├── popup.css         # 스타일
├── popup.js          # UI 로직 / 결과 표시 / 내보내기
├── content.js        # 페이지에 주입되는 추출 / 다음 버튼 클릭 도우미
├── background.js     # 다중 페이지 자동 순회 (service worker)
├── icons/            # 16/48/128 아이콘
└── README.md
```
